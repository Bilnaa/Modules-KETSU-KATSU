About KETSU

ESP: KETSU es la primera aplicación modular que permite a sus usuarios crear y cargar módulos para consumir cualquier tipo de contenido en internet, ya sean videos, imagenes e incluso recetas de cocina :)

ENG: KETSU is a modular app that allows the user to create and load modules for it, allowing the user to consume any kind of media imported by the user.

You can find the official page by this link: https://ketsu.app/

Documentation: https://appketsu.github.io/KETSU-Documentation/
