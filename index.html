What is KETSU? | ¿Qué es KETSU?

KETSU is a modular app that allows the user to create and load modules for it, allowing the user to consume any kind of media imported by the user.

Es la primera aplicación modular que permite a sus usuarios crear y cargar módulos para consumir cualquier tipo de contenido en internet, ya sean videos, imagenes e incluso recetas de cocina :)

Est une application modulaire qui permet à l'utilisateur de créer et de charger des modules pour elle, permettant à l'utilisateur de consommer tout type de média importé par l'utilisateur.

È un'app modulare che consente all'utente di creare e caricare moduli per essa, consentendo all'utente di consumare qualsiasi tipo di supporto importato dall'utente.

ユーザーがモジュールを作成してロードできるようにするモジュラーアプリであり、ユーザーがインポートしたあらゆる種類のメディアを消費できるようにします。

You can find the official page by this link: https://ketsu.app/
